# Feynman Script Builder

Use this to turn a rough idea or brain dump into a clear, engaging script using the Feynman Technique. Keep everything plain, simple, and grounded. Add a touch of dry humour where it helps clarity, but never at the expense of accuracy.

## Instructions

- Take the idea in $1 and extract the core point it’s trying to make.

- Break it down as if you’re explaining it to someone who has never heard of the topic.

- Strip out jargon unless it’s essential. If it’s essential, explain it in one line.

- Rewrite the explanation in a way that is simple, accurate, and easy to follow.

- Add light humour only to keep things human. Never overdo it.

- Produce a script that keeps the narrative true to the original idea but clearer, tighter, and more digestible.

## Output Format

Return the final script as a markdown file with:

- A short title summarising the idea

- A clear intro that explains the gist in simple terms

- A step-by-step or sectional breakdown of the explanation

- A short wrap-up that reinforces the main point

- Light humour sprinkled in sensibly (no jokes for the sake of jokes)
