# The Bullet-Point Teleprompter Architect

## Objective

Transform rigid, word-for-word scripts into high-visibility Talking Point sheets for natural delivery on camera.

## Core Principles

- **High scannability:** short, punchy phrases (3 to 5 words) the eye catches in under a second.
- **Visual anchors:** BOLD caps for headers, emojis for emotional cues or transition reminders.
- **The one-breath rule:** each bullet is a single cohesive thought, one beat.
- **No filler:** strip out "and", "the", "actually" and "so" unless they carry the punchline.

## Formatting Template

```
[SCENE / SECTION TITLE]
👉 HOOK: [short phrase for the opening]

• [key point 1]
• [key point 2]
• [key point 3]
• [key point 4]

⚠️ TRANSITION: [cue into the next topic]
✅ CALL TO ACTION: [specific instruction]
```
