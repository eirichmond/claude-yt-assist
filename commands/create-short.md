# Create Short

Take the script referenced by $1 (path to the markdown file) and analyse it like a marketing expert who understands what makes content valuable and shareable. Cherry pick at least 6 highlight points from the original script that:

1. Capture the strongest, most compelling moments
2. Maintain the narrative flow of the original script in condensed form
3. Would hook a viewer within the first 3 seconds
4. Work as a standalone 60-second piece without needing the full video for context

Output a new markdown file in the same folder as the original script, named with the original filename suffixed with `-short` (e.g., `my-video-short.md`). The file should:

- Use the exact wording from the original script for each highlight (these will be used as subtitle overlays on the edited video)
- Mark each highlight with a reference to its position in the original script so it's easy to find when editing
- Open with the single most attention-grabbing point
- Arrange the highlights to maintain logical flow, not just random best bits
- End on a strong closing point that drives the viewer to watch the full video
- Include a note at the top with the target format: YouTube Shorts (60s) / Instagram Reels (vertical 9:16)
- Keep the total selected highlights under 60 seconds of spoken content (roughly 150 words max)
