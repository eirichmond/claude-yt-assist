# Create Three Interstitials {script-path} {third-topic}

## Parameters

| Parameter | Required | Default | Description                              |
|-----------|----------|---------|------------------------------------------|
| $1        | Yes      | none    | path to the script markdown              |
| $2        | Yes      | none    | topic for the third interstitial         |

If either $1 or $2 is missing, ask for them before doing anything else.

Create three interstitials based on the script in $1.  
The first should prompt a like on YouTube for anyone finding the content useful.  
The second should encourage a subscription for more content in a similar vein.  
The third should be focused on $2, shaped to fit naturally with the surrounding narrative.

Place each interstitial at logical points in the $1 markdown file where a brief pause makes sense and the flow won’t feel disrupted. Keep the tone plain, clear, and in my usual straightforward style.
